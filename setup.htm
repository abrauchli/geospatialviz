Development findings

KineticJS KineticJS quickly proved to be slower than expected. Redrawing all with all points enabled is painfully slow on my 5yr old laptop. The stack inspection reveals that it seems to be an issue with the way grouping is handled.

FlotJS To ease development, the open Javascript graphing library Flot was used to draw charts. This greatly reduces the required code while still allowing every required features. It proved to be a great charting library with many options and great configurability. FlotJS depends on jQuery, also a reson why I chose this library.

jQuery As base javascript framework, jQuery is used. This makes it easy to capture and process events.
